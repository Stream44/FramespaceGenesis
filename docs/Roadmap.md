Visualization Libraries
---

- [x] [Cytoscape.js](https://js.cytoscape.org/) - Graph theory (network) library for visualisation and analysis
  - [ ] https://github.com/iVis-at-Bilkent/cytoscape.js-fcose
- [ ] [visjs](https://visjs.org/) - A dynamic, browser based visualization library.
- [ ] [d3js.org](https://d3js.org/) - The JavaScript library for bespoke data visualization
- [ ] [Sigma.js](https://github.com/jacomyal/sigma.js) - A JavaScript library aimed at visualizing graphs of thousands of nodes and edges
- [ ] [Paper.js](http://paperjs.org/) - An open source vector graphics scripting framework that runs on top of the HTML5 Canvas.
- [ ] https://github.com/GuntherRademacher/rr - RR is a generator of syntax diagrams, also known as railroad diagrams. It is a self-contained tool with both a browser-based GUI and a batch mode.
- [ ] https://github.com/bramp/js-sequence-diagrams - Generates UML sequence diagrams from simple text
- [ ] https://github.com/mermaid-js/mermaid - Generation of diagrams like flowcharts or sequence diagrams from text in a similar manner as markdown
